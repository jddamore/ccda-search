Criticality Observation [2.16.840.1.113883.10.20.22.4.145] view pdf

Description:

This observation represents the gravity of the potential risk for future life-threatening adverse reactions when exposed to a substance known to cause an adverse reaction in that individual. When the worst case result is assessed to have a life-threatening or organ system threatening potential, it is considered to be of high criticality.

Contained By: Contains:
Allergy - Intolerance Observation (V2)
Substance or Device Allergy - Intolerance Observation (V2)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-32921).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-32922).
  • SHALL contain exactly one [1..1] templateId (CONF:81-32918) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.145" (CONF:81-32923).
  • SHALL contain exactly one [1..1] code (CONF:81-32919).
    • This code SHALL contain exactly one [1..1] @code="82606-5" Criticality (CONF:81-32925).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:81-32926).
  • SHALL contain exactly one [1..1] statusCode (CONF:81-32920).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:81-32927).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHALL be selected from ValueSet Criticality Observation 2.16.840.1.113883.1.11.20549 DYNAMIC (CONF:81-32928).
