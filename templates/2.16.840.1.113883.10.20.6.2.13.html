Code Observations [2.16.840.1.113883.10.20.6.2.13] view pdf

Description:

DICOM Template 2000 specifies that Imaging Report Elements of Value Type Code are contained in sections. The Imaging Report Elements are inferred from Basic Diagnostic Imaging Report Observations that consist of image references and measurements (linear, area, volume, and numeric). Coded DICOM Imaging Report Elements in this context are mapped to CDA-coded observations that are section components and are related to the SOP Instance Observations (templateId 2.16.840.1.113883.10.20.6.2.8) or Quantity Measurement Observations (templateId 2.16.840.1.113883.10.20.6.2.14) by the SPRT (Support) act relationship.

Contained By: Contains:
Diagnostic Imaging Report (V3)
 SOP Instance Observation
Quantity Measurement Observation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-9304).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9305).
  • SHALL contain exactly one [1..1] templateId (CONF:81-15523).
    • This templateId SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.6.2.13" (CONF:81-15524).
  • SHALL contain exactly one [1..1] code (CONF:81-19181).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-9309).
  • SHALL contain exactly one [1..1] value (CONF:81-9308).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9311) such that it
    • SHALL contain exactly one [1..1] @typeCode="SPRT" Has Support (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9312).
    • SHALL contain exactly one [1..1] SOP Instance Observation (identifier: 2.16.840.1.113883.10.20.6.2.8) (CONF:81-16083).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9314) such that it
    • SHALL contain exactly one [1..1] @typeCode="SPRT" Has Support (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9315).
    • SHALL contain exactly one [1..1] Quantity Measurement Observation (identifier: 2.16.840.1.113883.10.20.6.2.14) (CONF:81-16084).

  • Code Observations SHALL be rendered into section/text in separate paragraphs (CONF:81-9310).
