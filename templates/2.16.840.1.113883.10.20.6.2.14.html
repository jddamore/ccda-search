Quantity Measurement Observation [2.16.840.1.113883.10.20.6.2.14] view pdf

Description:

A Quantity Measurement Observation records quantity measurements based on image data such as linear, area, volume, and numeric measurements. The codes in DIRQuantityMeasurementTypeCodes (ValueSet: 2.16.840.1.113883.11.20.9.29) are from the qualifier hierarchy of SNOMED CT and are not valid for observation/code according to the Term Info guidelines. These codes can be used for backwards compatibility, but going forward, codes from the observable entity hierarchy will be requested and used.

Contained By: Contains:

Diagnostic Imaging Report (V3)
 SOP Instance Observation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-9317).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9318).
  • SHALL contain exactly one [1..1] templateId (CONF:81-9319) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.6.2.14" (CONF:81-10532).
  • SHALL contain exactly one [1..1] code (CONF:81-9320).
    • This code SHOULD contain zero or one [0..1] @code, which SHOULD be selected from ValueSet DIRQuantityMeasurementTypeCodes 2.16.840.1.113883.11.20.9.29 DYNAMIC (CONF:81-19210).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-9326).
  • SHALL contain exactly one [1..1] value with @xsi:type="PQ" (CONF:81-9324).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9327) such that it
    • SHALL contain exactly one [1..1] @typeCode="SPRT" Has Support (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9328).
    • SHALL contain exactly one [1..1] SOP Instance Observation (identifier: 2.16.840.1.113883.10.20.6.2.8) (CONF:81-15916).
