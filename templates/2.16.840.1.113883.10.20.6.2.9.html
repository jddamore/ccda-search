Purpose of Reference Observation [2.16.840.1.113883.10.20.6.2.9] view pdf

Description:

A Purpose of Reference Observation describes the purpose of the DICOM composite object reference. Appropriate codes, such as externally defined DICOM codes, may be used to specify the semantics of the purpose of reference. When this observation is absent, it implies that the reason for the reference is unknown.

Contained By: Contains:
SOP Instance Observation
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-9264).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9265).
  • SHALL contain exactly one [1..1] templateId (CONF:81-9266) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.6.2.9" (CONF:81-10531).
  • SHALL contain exactly one [1..1] code (CONF:81-9267).
    • This code SHOULD contain zero or one [0..1] @code="ASSERTION" Assertion (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4 STATIC) (CONF:81-19208).

    • For backwards compatibility with the DICOM CMET, the code MAY be drawn from ValueSet 2.16.840.1.113883.11.20.9.28 DICOMPurposeOfReference DYNAMIC (CONF:81-19209).

  • SHOULD contain zero or one [0..1] value with @xsi:type="CD", where the code SHOULD be selected from ValueSet DICOMPurposeOfReference 2.16.840.1.113883.11.20.9.28 DYNAMIC (CONF:81-9273).
