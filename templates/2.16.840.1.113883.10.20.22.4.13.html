Procedure Activity Observation (V2) [2.16.840.1.113883.10.20.22.4.13, release 2014-06-09] view pdf

Description:

The common notion of procedure is broader than that specified by the HL7 Version 3 Reference Information Model (RIM). Therefore procedure templates can be represented with various RIM classes: act (e.g., dressing change), observation (e.g., EEG), procedure (e.g., splenectomy).

This template represents procedures that result in new information about the patient that cannot be classified as a procedure according to the HL7 RIM. Examples of these procedures are diagnostic imaging procedures, EEGs, and EKGs.

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-8282).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-8237).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-8238) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.13" (CONF:1098-10520).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32507).
  • SHALL contain at least one [1..*] id (CONF:1098-8239).
  • SHALL contain exactly one [1..1] code (CONF:1098-19197).
    • This code SHOULD contain zero or one [0..1] originalText (CONF:1098-19198).
      • The originalText, if present, SHOULD contain zero or one [0..1] reference (CONF:1098-19199).
        • The reference, if present, SHOULD contain zero or one [0..1] @value (CONF:1098-19200).

          • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:1098-19201).

    • This @code SHOULD be selected from LOINC (CodeSystem: 2.16.840.1.113883.6.1) or SNOMED CT (CodeSystem: 2.16.840.1.113883.6.96), and MAY be selected from CPT-4 (CodeSystem: 2.16.840.1.113883.6.12) or ICD10 PCS (CodeSystem: 2.16.840.1.113883.6.4) or CDT-2 (Code System: 2.16.840.1.113883.6.13) (CONF:1098-19202).

  • SHALL contain exactly one [1..1] statusCode (CONF:1098-8245).
    • This statusCode SHALL contain exactly one [1..1] @code, which SHALL be selected from ValueSet ProcedureAct statusCode 2.16.840.1.113883.11.20.9.22 STATIC 2014-04-23 (CONF:1098-32365).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-8246).
  • MAY contain zero or one [0..1] priorityCode, which SHALL be selected from ValueSet ActPriority 2.16.840.1.113883.1.11.16866 DYNAMIC (CONF:1098-8247).
  • SHALL contain exactly one [1..1] value (CONF:1098-16846).
    • This value MAY contain zero or one [0..1] @nullFlavor (CONF:1098-32778).
  • MAY contain zero or one [0..1] methodCode (CONF:1098-8248).

    • MethodCode SHALL NOT conflict with the method inherent in Observation / code (CONF:1098-8249).

  • SHOULD contain zero or more [0..*] targetSiteCode, which SHALL be selected from ValueSet Body Site Value Set 2.16.840.1.113883.3.88.12.3221.8.9 DYNAMIC (CONF:1098-8250).
  • SHOULD contain zero or more [0..*] performer (CONF:1098-8251).
    • The performer, if present, SHALL contain exactly one [1..1] assignedEntity (CONF:1098-8252).
      • This assignedEntity SHALL contain at least one [1..*] id (CONF:1098-8253).
      • This assignedEntity SHALL contain at least one [1..*] addr (CONF:1098-8254).
      • This assignedEntity SHALL contain at least one [1..*] telecom (CONF:1098-8255).
      • This assignedEntity SHOULD contain zero or one [0..1] representedOrganization (CONF:1098-8256).
        • The representedOrganization, if present, SHOULD contain zero or more [0..*] id (CONF:1098-8257).
        • The representedOrganization, if present, MAY contain zero or more [0..*] name (CONF:1098-8258).
        • The representedOrganization, if present, SHALL contain exactly one [1..1] telecom (CONF:1098-8260).
        • The representedOrganization, if present, SHALL contain exactly one [1..1] addr (CONF:1098-8259).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-32478).
  • MAY contain zero or more [0..*] participant (CONF:1098-8261) such that it
    • SHALL contain exactly one [1..1] @typeCode="LOC" Location (CodeSystem: HL7ParticipationType 2.16.840.1.113883.5.90 STATIC) (CONF:1098-8262).
    • SHALL contain exactly one [1..1] Service Delivery Location (identifier: 2.16.840.1.113883.10.20.22.4.32) (CONF:1098-15904).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8264) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8265).
    • SHALL contain exactly one [1..1] @inversionInd="true" true (CONF:1098-8266).
    • SHALL contain exactly one [1..1] encounter (CONF:1098-8267).
      • This encounter SHALL contain exactly one [1..1] @classCode="ENC" Encounter (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-8268).
      • This encounter SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-8269).
      • This encounter SHALL contain exactly one [1..1] id (CONF:1098-8270).

        • Set encounter/id to the id of an encounter in another section to signify they are the same encounter (CONF:1098-16847).

  • MAY contain zero or one [0..1] entryRelationship (CONF:1098-8272) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" Has Subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8273).
    • SHALL contain exactly one [1..1] @inversionInd="true" true (CONF:1098-8274).
    • SHALL contain exactly one [1..1] Instruction (V2) (identifier: 2.16.840.1.113883.10.20.22.4.20) (CONF:1098-31394).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8276) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" Has Reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8277).
    • SHALL contain exactly one [1..1] Indication (V2) (identifier: 2.16.840.1.113883.10.20.22.4.19) (CONF:1098-15906).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8279) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8280).
    • SHALL contain exactly one [1..1] Medication Activity (V2) (identifier: 2.16.840.1.113883.10.20.22.4.16) (CONF:1098-15907).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-32470) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-32471).
    • SHALL contain exactly one [1..1] Reaction Observation (V2) (identifier: 2.16.840.1.113883.10.20.22.4.9) (CONF:1098-32472).
