Discharge Medication (V3) [2.16.840.1.113883.10.20.22.4.35, release 2016-03-01] view pdf

Description:

This template represents medications that the patient is intended to take (or stop) after discharge.

Contained By: Contains:
Discharge Medications Section (entries optional) (V3)
Discharge Medications Section (entries required) (V3)
 Medication Activity (V2)

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="ACT" (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1198-7689).
  • SHALL contain exactly one [1..1] @moodCode="EVN" (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1198-7690).
  • SHALL contain exactly one [1..1] templateId (CONF:1198-16760) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.35" (CONF:1198-16761).
    • SHALL contain exactly one [1..1] @extension="2016-03-01" (CONF:1198-32513).
  • SHALL contain exactly one [1..1] code (CONF:1198-7691).
    • This code SHALL contain exactly one [1..1] @code="10183-2" Hospital discharge medication (CONF:1198-19161).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-32159).
    • This code SHALL contain exactly one [1..1] translation (CONF:1198-32952).
      • This translation SHALL contain exactly one [1..1] @code="75311-1" Discharge Medication (CONF:1198-32953).
      • This translation SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-32954).
  • SHALL contain exactly one [1..1] statusCode (CONF:1198-32779).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1198-32780).
  • SHALL contain at least one [1..*] entryRelationship (CONF:1198-7692) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" Has Subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1198-7693).
    • SHALL contain exactly one [1..1] Medication Activity (V2) (identifier: 2.16.840.1.113883.10.20.22.4.16) (CONF:1198-15525).
