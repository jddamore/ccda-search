Instructions Section (V2) [2.16.840.1.113883.10.20.22.2.45, release 2014-06-09] view pdf

Description:

The Instructions Section records instructions given to a patient. List patient decision aids here.

Contained By: Contains:
History and Physical (V3)
Progress Note (V3)
 Instruction (V2)

Conformance Statements:

  • MAY contain zero or one [0..1] @nullFlavor="NI" No information (CodeSystem: HL7NullFlavor 2.16.840.1.113883.5.1008) (CONF:1098-32835).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-10112) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.45" (CONF:1098-31384).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32599).
  • SHALL contain exactly one [1..1] code (CONF:1098-15375).
    • This code SHALL contain exactly one [1..1] @code="69730-0" Instructions (CONF:1098-15376).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1098-32148).
  • SHALL contain exactly one [1..1] title (CONF:1098-10114).
  • SHALL contain exactly one [1..1] text (CONF:1098-10115).
  • SHALL contain at least one [1..*] entry (CONF:1098-10116) such that it
    • SHALL contain exactly one [1..1] Instruction (V2) (identifier: 2.16.840.1.113883.10.20.22.4.20) (CONF:1098-31398).
