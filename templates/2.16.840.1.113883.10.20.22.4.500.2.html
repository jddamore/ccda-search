Care Team Type Observation [2.16.840.1.113883.10.20.22.4.500.2, release 2019-07-01] view pdf

Description:

This template is used to express the care team type. A care team can have multiple care team types. Examples include but are not limited to:

  • Condition focused, longitudinal care team
  • Event focused, Home & Community Based Services care team
  • Condition focused, clinical research care team
  • Public health focused, Longitudinal care-coordination care team
Contained By: Contains:
Care Team Organizer
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:4435-101).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:4435-102).
  • SHALL contain exactly one [1..1] templateId (CONF:4435-99) such that it
    • SHALL contain exactly one [1..1] @root=" 2.16.840.1.113883.10.20.22.4.500.2" (CONF:4435-106).
    • SHALL contain exactly one [1..1] @extension="2019-07-01" (CONF:4435-108).
  • SHALL contain exactly one [1..1] code (CONF:4435-97).
    • This code SHALL contain exactly one [1..1] @code="86744-0" Care Team (CONF:4435-103).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" LOINC (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:4435-104).
  • SHALL contain exactly one [1..1] statusCode (CONF:4435-100).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:4435-107).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD" (CONF:4435-98) such that it
    • SHALL contain exactly one [1..1] @code, which SHALL be selected from ValueSet Care Team Category 2.16.840.1.113883.4.642.3.155 DYNAMIC (CONF:4435-109).
