Goal Observation [2.16.840.1.113883.10.20.22.4.121] view pdf

Description:

This template represents a patient health goal. A Goal Observation template may have related components that are acts, encounters, observations, procedures, substance administrations, or supplies.

A goal may be a patient or provider goal. If the author is set to the recordTarget (patient), this is a patient goal. If the author is set to a provider, this is a provider goal. If both patient and provider are set as authors, this is a negotiated goal.

A goal usually has a related health concern and/or risk.

A goal may have components consisting of other goals (milestones). These milestones are related to the overall goal through entryRelationships.

  • SHALL contain at least one [1..*] id (CONF:1098-32332).
  • SHALL contain exactly one [1..1] code, which SHOULD be selected from CodeSystem LOINC (2.16.840.1.113883.6.1) (CONF:1098-30784).
  • SHALL contain exactly one [1..1] statusCode (CONF:1098-32333).
    • This statusCode SHALL contain exactly one [1..1] @code="active" (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1098-32334).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-32335).
  • MAY contain zero or one [0..1] value (CONF:1098-32743).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-30995).

    • SHALL contain exactly one [1..1] Entry Reference (identifier: 2.16.840.1.113883.10.20.22.4.122) (CONF:1098-30703).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-30704) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-30705).

  • SHOULD contain zero or one [0..1] entryRelationship (CONF:1098-30785) such that it
