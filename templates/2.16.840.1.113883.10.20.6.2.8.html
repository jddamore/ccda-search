SOP Instance Observation [observation, 2.16.840.1.113883.10.20.6.2.8, open] view pdf view prototype

Description:

A SOP Instance Observation contains the DICOM Service Object Pair (SOP) Instance information for referenced DICOM composite objects. The SOP Instance act class is used to reference both image and non-image DICOM instances. The text attribute contains the DICOM WADO reference.

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="DGIMG" Diagnostic Image (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6  STATIC) (CONF:81-9240).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9241).
  • SHALL contain at least one [1..*] id (CONF:81-9242).
  • SHALL contain exactly one [1..1] code (CONF:81-9244).
    • This code SHALL contain exactly one [1..1] @code (CONF:81-19225).

      • @code is an OID for a valid SOP class name UID (CONF:81-19226).

    • This code SHALL contain exactly one [1..1] @codeSystem="1.2.840.10008.2.6.1" DCMUID (CONF:81-19227).
  • SHOULD contain zero or one [0..1] text (CONF:81-9246).
    • The text, if present, SHALL contain exactly one [1..1] @mediaType="application/dicom" (CONF:81-9247).
    • The text, if present, SHALL contain exactly one [1..1] reference (CONF:81-9248).

      • SHALL contain a @value that contains a WADO reference as a URI (CONF:81-9249).

  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-9250).
    • The effectiveTime, if present, SHALL contain exactly one [1..1] @value (CONF:81-9251).
    • The effectiveTime, if present, SHALL NOT contain [0..0] low (CONF:81-9252).
    • The effectiveTime, if present, SHALL NOT contain [0..0] high (CONF:81-9253).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9254) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" Has Subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9255).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9257) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" Has Reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9258).
    • SHALL contain exactly one [1..1] Purpose of Reference Observation (identifier: 2.16.840.1.113883.10.20.6.2.9) (CONF:81-15935).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9260) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9261).
    • SHALL contain exactly one [1..1] Referenced Frames Observation (identifier: 2.16.840.1.113883.10.20.6.2.10) (CONF:81-15936).

    • This entryRelationship SHALL be present if the referenced DICOM object is a multiframe object and the reference does not apply to all frames (CONF:81-9263).
