Series Act [2.16.840.1.113883.10.20.22.4.63] view pdf

Description:

A Series Act contains the DICOM series information for referenced DICOM composite objects. The series information defines the attributes that are used to group composite instances into distinct logical sets. Each series is associated with exactly one study. Series Act clinical statements are only instantiated in the DICOM Object Catalog section inside a Study Act, and thus do not require a separate templateId; in other sections, the SOP Instance Observation is included directly.

Contained By: Contains:
Study Act
 SOP Instance Observation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-9222).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9223).
  • SHALL contain exactly one [1..1] templateId (CONF:81-10918) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.63" (CONF:81-10919).
  • SHALL contain at least one [1..*] id (CONF:81-9224).
    • Such ids SHALL contain exactly one [1..1] @root (CONF:81-9225).
    • Such ids SHALL NOT contain [0..0] @extension (CONF:81-9226).
  • SHALL contain exactly one [1..1] code (CONF:81-19166).
    • This code SHALL contain exactly one [1..1] @code="113015" (CONF:81-19167).
    • This code MAY contain zero or one [0..1] @codeSystem="1.2.840.10008.2.16.4" (CodeSystem: DCM 1.2.840.10008.2.16.4) (CONF:81-26461).
    • This code SHALL contain exactly one [1..1] qualifier (CONF:81-26462).
      • This qualifier SHALL contain exactly one [1..1] name (CONF:81-26463).
        • This name SHALL contain exactly one [1..1] @code="121139" Modality (CONF:81-26464).
        • This name SHALL contain exactly one [1..1] @codeSystem="1.2.840.10008.2.16.4" (CodeSystem: DCM 1.2.840.10008.2.16.4) (CONF:81-26465).
      • This qualifier SHALL contain exactly one [1..1] value (CONF:81-26466).
  • MAY contain zero or one [0..1] text (CONF:81-9233).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-9235).
  • SHALL contain at least one [1..*] entryRelationship (CONF:81-9237) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9238).
    • SHALL contain exactly one [1..1] SOP Instance Observation (identifier: 2.16.840.1.113883.10.20.6.2.8) (CONF:81-15927).
