Discharge Diagnosis Section (V3) [2.16.840.1.113883.10.20.22.2.24, release 2015-08-01] view pdf

Description:

This template represents problems or diagnoses present at the time of discharge which occurred during the hospitalization. This section includes an optional entry to record patient diagnoses specific to this visit. Problems that need ongoing tracking should also be included in the Problem Section.

Contained By: Contains:
Discharge Summary (V3)
Transfer Summary (V2)
 Hospital Discharge Diagnosis (V3)

Conformance Statements:

  • SHALL contain exactly one [1..1] templateId (CONF:1198-7979) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.24" (CONF:1198-10394).
    • SHALL contain exactly one [1..1] @extension="2015-08-01" (CONF:1198-32549).
  • SHALL contain exactly one [1..1] code (CONF:1198-15355).
    • This code SHALL contain exactly one [1..1] @code="11535-2" Hospital Discharge Diagnosis (CodeSystem: LOINC 2.16.840.1.113883.6.1 STATIC) (CONF:1198-15356).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-30861).
    • This code SHALL contain exactly one [1..1] translation (CONF:1198-32834) such that it
      • SHALL contain exactly one [1..1] @code="78375-3" Discharge Diagnosis (CONF:1198-32835).
      • SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-32836).
  • SHALL contain exactly one [1..1] title (CONF:1198-7981).
  • SHALL contain exactly one [1..1] text (CONF:1198-7982).
  • SHOULD contain zero or one [0..1] entry (CONF:1198-7983).
    • The entry, if present, SHALL contain exactly one [1..1] Hospital Discharge Diagnosis (V3) (identifier: 2.16.840.1.113883.10.20.22.4.33) (CONF:1198-15489).
