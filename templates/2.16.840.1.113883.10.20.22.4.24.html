Drug Vehicle [2.16.840.1.113883.10.20.22.4.24] view pdf

Description:

This template represents the vehicle (e.g., saline, dextrose) for administering a medication.

Contained By: Contains:
Medication Activity (V2)
Immunization Activity (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="MANU" (CodeSystem: HL7RoleClass 2.16.840.1.113883.5.110 STATIC) (CONF:81-7490).
  • SHALL contain exactly one [1..1] templateId (CONF:81-7495) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.24" (CONF:81-10493).
  • SHALL contain exactly one [1..1] code (CONF:81-19137).
    • This code SHALL contain exactly one [1..1] @code="412307009" Drug Vehicle (CONF:81-19138).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.96" (CodeSystem: SNOMED CT 2.16.840.1.113883.6.96) (CONF:81-26502).
  • SHALL contain exactly one [1..1] playingEntity (CONF:81-7492).
    • This playingEntity SHALL contain exactly one [1..1] code (CONF:81-7493).
    • This playingEntity MAY contain zero or one [0..1] name (CONF:81-7494).

      • This playingEntity/name MAY be used for the vehicle name in text, such as Normal Saline (CONF:81-10087).
