Procedure Activity Act (V2) [2.16.840.1.113883.10.20.22.4.12, release 2014-06-09] view pdf

Description:

This template represents any act that cannot be classified as an observation or procedure according to the HL7 RIM. Examples of these acts are a dressing change, teaching or feeding a patient, or providing comfort measures.The common notion of "procedure" is broader than that specified by the HL7 Version 3 Reference Information Model (RIM). Procedure templates can be represented with various RIM classes: act (e.g., dressing change), observation (e.g., EEG), procedure (e.g., splenectomy).

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-8289).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-8290).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-8291) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.12" (CONF:1098-10519).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32505).
  • SHALL contain at least one [1..*] id (CONF:1098-8292).
  • SHALL contain exactly one [1..1] code (CONF:1098-8293).
    • This code SHOULD contain zero or one [0..1] originalText (CONF:1098-19186).
      • The originalText, if present, MAY contain zero or one [0..1] reference (CONF:1098-19187).
        • The reference, if present, MAY contain zero or one [0..1] @value (CONF:1098-19188).

          • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:1098-19189).

    • This @code SHOULD be selected from LOINC (CodeSystem: 2.16.840.1.113883.6.1) or SNOMED CT (CodeSystem: 2.16.840.1.113883.6.96), and MAY be selected from CPT-4 (CodeSystem: 2.16.840.1.113883.6.12) or ICD10 PCS (CodeSystem: 2.16.840.1.113883.6.4) or CDT-2 (Code System: 2.16.840.1.113883.6.13) (CONF:1098-19190).

  • SHALL contain exactly one [1..1] statusCode (CONF:1098-8298).
    • This statusCode SHALL contain exactly one [1..1] @code, which SHALL be selected from ValueSet ProcedureAct statusCode 2.16.840.1.113883.11.20.9.22 STATIC 2014-04-23 (CONF:1098-32364).
  • SHALL contain exactly one [1..1] effectiveTime (CONF:1098-8299).
  • MAY contain zero or one [0..1] priorityCode, which SHALL be selected from ValueSet ActPriority 2.16.840.1.113883.1.11.16866 DYNAMIC (CONF:1098-8300).
  • SHOULD contain zero or more [0..*] performer (CONF:1098-8301).
    • The performer, if present, SHALL contain exactly one [1..1] assignedEntity (CONF:1098-8302).
      • This assignedEntity SHALL contain at least one [1..*] id (CONF:1098-8303).
      • This assignedEntity SHALL contain at least one [1..*] addr (CONF:1098-8304).
      • This assignedEntity SHALL contain at least one [1..*] telecom (CONF:1098-8305).
      • This assignedEntity SHOULD contain zero or one [0..1] representedOrganization (CONF:1098-8306).
        • The representedOrganization, if present, SHOULD contain zero or more [0..*] id (CONF:1098-8307).
        • The representedOrganization, if present, MAY contain zero or more [0..*] name (CONF:1098-8308).
        • The representedOrganization, if present, SHALL contain at least one [1..*] telecom (CONF:1098-8310).
        • The representedOrganization, if present, SHALL contain at least one [1..*] addr (CONF:1098-8309).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-32477).
  • MAY contain zero or more [0..*] participant (CONF:1098-8311) such that it
    • SHALL contain exactly one [1..1] @typeCode="LOC" Location (CodeSystem: HL7ParticipationType 2.16.840.1.113883.5.90 STATIC) (CONF:1098-8312).
    • SHALL contain exactly one [1..1] Service Delivery Location (identifier: 2.16.840.1.113883.10.20.22.4.32) (CONF:1098-15599).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8314) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8315).
    • SHALL contain exactly one [1..1] @inversionInd="true" true (CONF:1098-8316).
    • SHALL contain exactly one [1..1] encounter (CONF:1098-8317).
      • This encounter SHALL contain exactly one [1..1] @classCode="ENC" Encounter (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-8318).
      • This encounter SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-8319).
      • This encounter SHALL contain exactly one [1..1] id (CONF:1098-8320).

        • Set the encounter ID to the ID of an encounter in another section to signify they are the same encounter (CONF:1098-16849).

  • MAY contain zero or one [0..1] entryRelationship (CONF:1098-8322) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" Has Subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8323).
    • SHALL contain exactly one [1..1] @inversionInd="true" true (CONF:1098-8324).
    • SHALL contain exactly one [1..1] Instruction (V2) (identifier: 2.16.840.1.113883.10.20.22.4.20) (CONF:1098-31396).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8326) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" Has Reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8327).
    • SHALL contain exactly one [1..1] Indication (V2) (identifier: 2.16.840.1.113883.10.20.22.4.19) (CONF:1098-15601).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-8329) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" Has Component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-8330).
    • SHALL contain exactly one [1..1] Medication Activity (V2) (identifier: 2.16.840.1.113883.10.20.22.4.16) (CONF:1098-15602).
