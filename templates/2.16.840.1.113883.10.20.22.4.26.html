Vital Signs Organizer (V3) [2.16.840.1.113883.10.20.22.4.26, release 2015-08-01] view pdf

Description:

This template provides a mechanism for grouping vital signs (e.g., grouping systolic blood pressure and diastolic blood pressure).

Contained By: Contains:
Vital Signs Section (entries optional) (V3)
Vital Signs Section (entries required) (V3)
 Vital Sign Observation (V2)
Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="CLUSTER" CLUSTER (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1198-7279).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1198-7280).
  • SHALL contain exactly one [1..1] templateId (CONF:1198-7281) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.26" (CONF:1198-10528).
    • SHALL contain exactly one [1..1] @extension="2015-08-01" (CONF:1198-32582).
  • SHALL contain at least one [1..*] id (CONF:1198-7282).
  • SHALL contain exactly one [1..1] code (CONF:1198-32740).
    • This code SHALL contain exactly one [1..1] @code="46680005" Vital Signs (CONF:1198-32741).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.96" SNOMED CT (CodeSystem: SNOMED CT 2.16.840.1.113883.6.96) (CONF:1198-32742).
    • This code SHALL contain exactly one [1..1] translation (CONF:1198-32743) such that it
      • SHALL contain exactly one [1..1] @code="74728-7" Vital signs, weight, height, head circumference, oximetry, BMI, and BSA panel - HL7.CCDAr1.1 (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-32744).
      • SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" LOINC (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-32746).
  • SHALL contain exactly one [1..1] statusCode (CONF:1198-7284).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1198-19120).
  • SHALL contain exactly one [1..1] effectiveTime (CONF:1198-7288).
    Note: The effectiveTime may be a timestamp or an interval that spans the effectiveTimes of the contained vital signs observations.
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1198-31153).
  • SHALL contain at least one [1..*] component (CONF:1198-7285) such that it
    • SHALL contain exactly one [1..1] Vital Sign Observation (V2) (identifier: 2.16.840.1.113883.10.20.22.4.27) (CONF:1198-15946).
