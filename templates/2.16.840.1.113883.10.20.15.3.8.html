Pregnancy Observation [2.16.840.1.113883.10.20.15.3.8] view pdf

Description:

This clinical statement represents current and/or prior pregnancy dates enabling investigators to determine if the subject of the case report was pregnant during the course of a condition.

Contained By: Contains:
Health Concern Act (V2)
Risk Concern Act (V2)
Social History Section (V3)
 Estimated Date of Delivery

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-451).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-452).
  • SHALL contain exactly one [1..1] templateId (CONF:81-16768) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.15.3.8" (CONF:81-16868).
  • SHALL contain exactly one [1..1] code (CONF:81-19153).
    • This code SHALL contain exactly one [1..1] @code="ASSERTION" Assertion (CONF:81-19154).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.5.4" (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:81-26505).
  • SHALL contain exactly one [1..1] statusCode (CONF:81-455).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:81-19110).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-2018).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHALL be selected from ValueSet Extended Pregnancy Status 2.16.840.1.113762.1.4.1099.24 DYNAMIC (CONF:81-457).
  • MAY contain zero or one [0..1] entryRelationship (CONF:81-458) such that it
    • SHALL contain exactly one [1..1] @typeCode="REFR" Refers to (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-459).
    • SHALL contain exactly one [1..1] Estimated Date of Delivery (identifier: 2.16.840.1.113883.10.20.15.3.1) (CONF:81-15584).
