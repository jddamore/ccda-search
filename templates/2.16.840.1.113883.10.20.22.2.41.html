Hospital Discharge Instructions Section [2.16.840.1.113883.10.20.22.2.41] view pdf

Description:

The Hospital Discharge Instructions Section records instructions at discharge.

Contained By: Contains:
Discharge Summary (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] templateId (CONF:81-9919) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.41" (CONF:81-10395).
  • SHALL contain exactly one [1..1] code (CONF:81-15357).
    • This code SHALL contain exactly one [1..1] @code="8653-8" Hospital Discharge Instructions (CONF:81-15358).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:81-26481).
  • SHALL contain exactly one [1..1] title (CONF:81-9921).
  • SHALL contain exactly one [1..1] text (CONF:81-9922).
