Planned Procedure Section (V2) [2.16.840.1.113883.10.20.22.2.30, release 2014-06-09] view pdf

Description:

This section contains the procedure(s) that a clinician planned based on the preoperative assessment.

Contained By: Contains:
Procedure Note (V3)
Operative Note (V3)
 Planned Procedure (V2)

Conformance Statements:

  • SHALL contain exactly one [1..1] templateId (CONF:1098-8082) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.30" (CONF:1098-10436).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32590).
  • SHALL contain exactly one [1..1] code (CONF:1098-15399).
    • This code SHALL contain exactly one [1..1] @code="59772-4" Planned Procedure (CONF:1098-15400).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1098-32151).
  • SHALL contain exactly one [1..1] title (CONF:1098-8084).
  • SHALL contain exactly one [1..1] text (CONF:1098-8085).
  • MAY contain zero or more [0..*] entry (CONF:1098-8744) such that it
    • SHALL contain exactly one [1..1] Planned Procedure (V2) (identifier: 2.16.840.1.113883.10.20.22.4.41) (CONF:1098-15502).
