Comment Activity [2.16.840.1.113883.10.20.22.4.64] view pdf

Description:

Comments are free text data that cannot otherwise be recorded using data elements already defined by this specification. They are not to be used to record information that can be recorded elsewhere. For example, a free text description of the severity of an allergic reaction would not be recorded in a comment.

Contained By: Contains:
none Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="ACT" Act (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-9425).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9426).
  • SHALL contain exactly one [1..1] templateId (CONF:81-9427) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.64" (CONF:81-10491).
  • SHALL contain exactly one [1..1] code (CONF:81-9428).
    • This code SHALL contain exactly one [1..1] @code="48767-8" Annotation Comment (CONF:81-19159).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:81-26501).
  • SHALL contain exactly one [1..1] text (CONF:81-9430).
    • This text SHALL contain exactly one [1..1] reference (CONF:81-15967).
      • This reference SHALL contain exactly one [1..1] @value (CONF:81-15968).

        • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:81-15969).

    • This text SHALL contain exactly one [1..1] reference/@value (CONF:81-9431).
  • SHOULD contain zero or one [0..1] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:81-9433).

  • Data elements defined elsewhere in the specification SHALL NOT be recorded using the Comment Activity (CONF:81-9429).
