Advance Directive Organizer (V2) [2.16.840.1.113883.10.20.22.4.108, release 2015-08-01] view pdf

Description:

This clinical statement groups a set of advance directive observations.

Contained By: Contains:
Advance Directives Section (entries optional) (V3)
Advance Directives Section (entries required) (V3)
 Advance Directive Observation (V3)
Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="CLUSTER" Cluster (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6) (CONF:1198-28410).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1198-28411).
  • SHALL contain exactly one [1..1] templateId (CONF:1198-28412) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.108" (CONF:1198-28413).
    • SHALL contain exactly one [1..1] @extension="2015-08-01" (CONF:1198-32876).
  • SHALL contain at least one [1..*] id (CONF:1198-28414).
  • SHALL contain exactly one [1..1] code (CONF:1198-28415).
    • This code SHALL contain exactly one [1..1] @code="45473-6" Advance directive - living will (CONF:1198-31230).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1 STATIC) (CONF:1198-31231).
  • SHALL contain exactly one [1..1] statusCode (CONF:1198-28418).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1198-31346).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1198-32407).
  • SHALL contain at least one [1..*] component (CONF:1198-28420) such that it
    • SHALL contain exactly one [1..1] Advance Directive Observation (V3) (identifier: 2.16.840.1.113883.10.20.22.4.48) (CONF:1198-28421).
