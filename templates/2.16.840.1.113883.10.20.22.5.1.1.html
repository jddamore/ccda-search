US Realm Person Name (PN.US.FIELDED) [2.16.840.1.113883.10.20.22.5.1.1] view pdf

Description:

The US Realm Clinical Document Person Name datatype flavor is a set of reusable constraints that can be used for Persons.

Contained By: Contains:
Drug Monitoring Act
Physician of Record Participant (V2)
Advance Directive Observation (V3)
Care Plan (V2)
Referral Note (V2)
US Realm Header (V3)
Diagnostic Imaging Report (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] name (CONF:81-9368).

    • The content of name SHALL be either a conformant Patient Name (PTN.US.FIELDED), or a string (CONF:81-9371).

    • The string SHALL NOT contain name parts (CONF:81-9372).
