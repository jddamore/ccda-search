Wound Characteristic [2.16.840.1.113883.10.20.22.4.134] view pdf

Description:

This template represents characteristics of a wound (e.g., integrity of suture line, odor, erythema).

Contained By: Contains:
Longitudinal Care Wound Observation (V2)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-29938).
  • SHALL contain exactly one [1..1] @moodCode="EVN" (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-29939).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-29940) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.134" (CONF:1098-29941).
  • SHALL contain at least one [1..*] id (CONF:1098-29942).
  • SHALL contain exactly one [1..1] code (CONF:1098-29943).
    • This code SHALL contain exactly one [1..1] @code="ASSERTION" assertion (CONF:1098-31540).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.5.4" (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:1098-31541).
  • SHALL contain exactly one [1..1] statusCode="completed" (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1098-29944).
  • SHALL contain exactly one [1..1] effectiveTime (CONF:1098-29946).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHALL be selected from ValueSet Wound Characteristic 2.16.840.1.113883.11.20.9.58 DYNAMIC (CONF:1098-29947).
