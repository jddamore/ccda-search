Nutrition Assessment [2.16.840.1.113883.10.20.22.4.138] view pdf

Description:

This template represents the patient's nutrition abilities and habits including intake, diet requirements or diet followed.

Contained By: Contains:
Nutritional Status Observation
Health Concern Act (V2)
Risk Concern Act (V2)
 Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-32914).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-32915).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-32916) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.138" (CONF:1098-32917).
  • SHALL contain at least one [1..*] id (CONF:1098-32918).
  • SHALL contain exactly one [1..1] code (CONF:1098-32919).
    • This code SHALL contain exactly one [1..1] @code="75303-8" Nutrition assessment (CONF:1098-32926).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1098-32927).
  • SHALL contain exactly one [1..1] statusCode (CONF:1098-32920).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1098-32921).
  • SHALL contain exactly one [1..1] effectiveTime (CONF:1098-32923).
  • SHALL contain exactly one [1..1] value (CONF:1098-32922).
