Patient Referral Act [2.16.840.1.113883.10.20.22.4.140] view pdf

Description:

This template represents the type of referral (e.g., for dental care, to a specialist, for aging problems) and represents whether the referral is for full care or shared care. It may contain a reference to another act in the document instance representing the clinical reason for the referral (e.g., problem, concern, procedure).

Contained By: Contains:
Reason for Referral Section (V2)
 Author Participation
Indication (V2)

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="PCPR" provision of care (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-30884).
  • SHALL contain exactly one [1..1] @moodCode, which SHALL be selected from ValueSet Patient Referral Act moodCode 2.16.840.1.113883.11.20.9.66 STATIC 2014-09-01 (CONF:1098-30885).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-30886) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.140" (CONF:1098-30887).
  • SHALL contain at least one [1..*] id (CONF:1098-30888).
  • SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet Referral Types 2.16.840.1.113883.11.20.9.56 DYNAMIC (CONF:1098-30889).
  • SHALL contain exactly one [1..1] statusCode (CONF:1098-30892).
    • This statusCode SHALL contain exactly one [1..1] @code="active" Active (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1098-31598).
  • SHALL contain exactly one [1..1] effectiveTime (CONF:1098-30893).
  • SHOULD contain zero or one [0..1] priorityCode (CONF:1098-32623).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-31612).
  • MAY contain zero or more [0..*] participant (CONF:1098-32635).
    • The participant, if present, SHALL contain exactly one [1..1] @typeCode="REFT" Referred to (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-32638).
    • The participant, if present, SHALL contain exactly one [1..1] participantRole (CONF:1098-32636).
      • This participantRole MAY contain zero or one [0..1] code, which SHOULD be selected from ValueSet Healthcare Provider Taxonomy 2.16.840.1.114222.4.11.1066 DYNAMIC (CONF:1098-32637).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-31604) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" has subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-31613).
    • SHALL contain exactly one [1..1] observation (CONF:1098-31605).
      • This observation SHALL contain exactly one [1..1] @classCode="OBS" observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6) (CONF:1098-31606).
      • This observation SHALL contain exactly one [1..1] @moodCode="RQO" request (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001) (CONF:1098-31607).
      • This observation SHALL contain exactly one [1..1] code (CONF:1098-31608).
        • This code SHALL contain exactly one [1..1] @code="ASSERTION" assertion (CONF:1098-31619).
        • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.5.4 " (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:1098-31620).
      • This observation SHALL contain exactly one [1..1] statusCode (CONF:1098-31614).
        • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1098-31615).
      • This observation SHOULD contain zero or one [0..1] priorityCode, which SHOULD be selected from ValueSet ActPriority 2.16.840.1.113883.1.11.16866 DYNAMIC (CONF:1098-32443).
      • This observation SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHOULD be selected from ValueSet Care Model 2.16.840.1.113883.11.20.9.61 DYNAMIC (CONF:1098-31611).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-31635) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" has reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-31636).
    • SHALL contain exactly one [1..1] Indication (V2) (identifier: 2.16.840.1.113883.10.20.22.4.19) (CONF:1098-32634).
