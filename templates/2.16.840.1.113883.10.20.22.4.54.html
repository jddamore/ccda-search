Immunization Medication Information (V2) [2.16.840.1.113883.10.20.22.4.54, release 2014-06-09] view pdf

Description:

The Immunization Medication Information represents product information about the immunization substance. The vaccine manufacturer and vaccine lot number are typically recorded in the medical record and should be included if known.

Contained By: Contains:
Planned Supply (V2)
Medication Supply Order (V2)
Medication Dispense (V2)
Planned Immunization Activity
Immunization Activity (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="MANU" (CodeSystem: HL7RoleClass 2.16.840.1.113883.5.110 STATIC) (CONF:1098-9002).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-9004) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.54" (CONF:1098-10499).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32602).
  • MAY contain zero or more [0..*] id (CONF:1098-9005).
  • SHALL contain exactly one [1..1] manufacturedMaterial (CONF:1098-9006).
    • This manufacturedMaterial SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet CVX Vaccines Administered Vaccine Set 2.16.840.1.113762.1.4.1010.6 DYNAMIC (CONF:1098-9007).
      • This code MAY contain zero or more [0..*] translation, which MAY be selected from ValueSet Vaccine Clinical Drug 2.16.840.1.113762.1.4.1010.8 DYNAMIC (CONF:1098-31543).
      • This code MAY contain zero or more [0..*] translation (CONF:1098-31881).
    • This manufacturedMaterial SHOULD contain zero or one [0..1] lotNumberText (CONF:1098-9014).
  • SHOULD contain zero or one [0..1] manufacturerOrganization (CONF:1098-9012).
