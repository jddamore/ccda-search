Medication Free Text Sig [2.16.840.1.113883.10.20.22.4.147] view pdf

Description:

The template is available to explicitly identify the free text Sig within each medication.

An example free text sig: Thyroxin 150 ug, take one tab by mouth every morning.

Contained By: Contains:
Medication Activity (V2)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="SBADM" (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:81-32770).
  • SHALL contain exactly one [1..1] @moodCode, which SHALL be selected from ValueSet MoodCodeEvnInt 2.16.840.1.113883.11.20.9.18 STATIC 2011-04-03 (CONF:81-32771).
    Note: moodCode must match the parent substanceAdministration EVN or INT
  • SHALL contain exactly one [1..1] templateId (CONF:81-32753) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.147" (CONF:81-32772).
  • SHALL contain exactly one [1..1] code (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:81-32775).
    • This code SHALL contain exactly one [1..1] @code="76662-6" Instructions Medication (CONF:81-32780).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1 STATIC) (CONF:81-32781).
  • SHALL contain exactly one [1..1] text (CONF:81-32754).
    • This text SHALL contain exactly one [1..1] reference (CONF:81-32755).
      • This reference SHOULD contain zero or one [0..1] @value (CONF:81-32756).

        • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:81-32774).

  • SHALL contain exactly one [1..1] consumable (CONF:81-32776).
    • This consumable SHALL contain exactly one [1..1] manufacturedProduct (CONF:81-32777).
      • This manufacturedProduct SHALL contain exactly one [1..1] manufacturedLabeledDrug (CONF:81-32778).
        • This manufacturedLabeledDrug SHALL contain exactly one [1..1] @nullFlavor="NA" Not Applicable (CONF:81-32779).
