Functional Status Problem Observation (DEPRECATED) [2.16.840.1.113883.10.20.22.4.68, release 2014-06-09] view pdf

Description:

A functional status problem observation is a clinical statement that represents a patient's functional perfomance and ability.

THIS TEMPLATE HAS BEEN DEPRECATED IN C-CDA R2 AND MAY BE DELETED FROM A FUTURE RELEASE OF THIS IMPLEMENTATION GUIDE. USE OF THIS TEMPLATE IS NOT RECOMMENDED.

Reason for deprecation: Functional Status Problem Observation has been merged, without loss of expressivity, into Functional Status Observation (2.16.840.1.113883.10.20.22.4.67).

Contained By: Contains:
Functional Status Section (V2)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-14282).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-14283).
  • MAY contain zero or one [0..1] @negationInd (CONF:1098-14307).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-14312) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.68" (CONF:1098-14313).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32601).
  • SHALL contain at least one [1..*] id (CONF:1098-14284).
  • SHALL contain exactly one [1..1] code (CONF:1098-14314).
    • This code SHOULD contain zero or one [0..1] @code="248536006" finding of functional performance and activity (CodeSystem: SNOMED CT 2.16.840.1.113883.6.96 STATIC) (CONF:1098-14315).
  • SHOULD contain zero or one [0..1] text (CONF:1098-14304).
    • The text, if present, SHOULD contain zero or one [0..1] reference (CONF:1098-15552).
      • The reference, if present, SHOULD contain zero or one [0..1] @value (CONF:1098-15553).

        • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:1098-15554).

  • SHALL contain exactly one [1..1] statusCode (CONF:1098-14286).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1098-19100).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-14287).
    • The effectiveTime, if present, SHALL contain exactly one [1..1] low (CONF:1098-26456).
    • The effectiveTime, if present, MAY contain zero or one [0..1] high (CONF:1098-26457).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHOULD be selected from ValueSet Problem 2.16.840.1.113883.3.88.12.3221.7.4 DYNAMIC (CONF:1098-14291).
    • This value MAY contain zero or one [0..1] @nullFlavor (CONF:1098-14292).

  • MAY contain zero or one [0..1] methodCode (CONF:1098-14316).
