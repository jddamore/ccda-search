Mental Status Organizer (V3) [2.16.840.1.113883.10.20.22.4.75, release 2015-08-01] view pdf

Description:

The Mental Status Organizer template may be used to group related Mental Status Observations (e.g., results of mental tests) and associated Assessment Scale Observations into subcategories and/or groupings by time. Subcategories can be things such as Mood and Affect, Behavior, Thought Process, Perception, Cognition, etc.

Contained By: Contains:
Mental Status Section (V2)
 Mental Status Observation (V3)

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="CLUSTER" Cluster (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6) (CONF:1198-14369).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1198-14371).
  • SHALL contain exactly one [1..1] templateId (CONF:1198-14375) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.75" (CONF:1198-14376).
    • SHALL contain exactly one [1..1] @extension="2015-08-01" (CONF:1198-32566).
  • SHALL contain at least one [1..*] id (CONF:1198-14377).
  • SHALL contain exactly one [1..1] code (CONF:1198-14378).
    • This code SHALL contain exactly one [1..1] @code (CONF:1198-14697).

      • SHOULD be selected from ICF (codeSystem 2.16.840.1.113883.6.254) OR LOINC (codeSystem 2.16.840.1.113883.6.96) (CONF:1198-14698).

  • SHALL contain exactly one [1..1] statusCode (CONF:1198-14372).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1198-19093).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:1198-32424).

    • The Organizer SHALL have at least one of code or effectiveTime (CONF:1198-32426).

  • SHALL contain at least one [1..*] component (CONF:1198-14373) such that it
    • SHALL contain exactly one [1..1] Mental Status Observation (V3) (identifier: 2.16.840.1.113883.10.20.22.4.74) (CONF:1198-14381).
