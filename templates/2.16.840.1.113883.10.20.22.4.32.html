Service Delivery Location [2.16.840.1.113883.10.20.22.4.32] view pdf

Description:

This clinical statement represents the location of a service event where an act, observation or procedure took place.

Contained By: Contains:
Procedure Activity Act (V2)
Procedure Activity Procedure (V2)
Procedure Activity Observation (V2)
Planned Encounter (V2)
Encounter Activity (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="SDLOC" (CodeSystem: HL7RoleCode 2.16.840.1.113883.5.111 STATIC) (CONF:81-7758).
  • SHALL contain exactly one [1..1] templateId (CONF:81-7635) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.32" (CONF:81-10524).
  • SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet HealthcareServiceLocation 2.16.840.1.113883.1.11.20275 DYNAMIC (CONF:81-16850).
  • SHOULD contain zero or more [0..*] addr (CONF:81-7760).
  • SHOULD contain zero or more [0..*] telecom (CONF:81-7761).
  • MAY contain zero or one [0..1] playingEntity (CONF:81-7762).
    • The playingEntity, if present, SHALL contain exactly one [1..1] @classCode="PLC" (CodeSystem: HL7EntityClass 2.16.840.1.113883.5.41 STATIC) (CONF:81-7763).
    • The playingEntity, if present, MAY contain zero or one [0..1] name (CONF:81-16037).
