Preoperative Diagnosis Section (V3) [2.16.840.1.113883.10.20.22.2.34, release 2015-08-01] view pdf

Description:

The Preoperative Diagnosis Section records the surgical diagnoses assigned to the patient before the surgical procedure which are the reason for the surgery. The preoperative diagnosis is, in the surgeon's opinion, the diagnosis that will be confirmed during surgery.

Contained By: Contains:
Operative Note (V3)
 Preoperative Diagnosis (V3)

Conformance Statements:

  • SHALL contain exactly one [1..1] templateId (CONF:1198-8097) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.2.34" (CONF:1198-10439).
    • SHALL contain exactly one [1..1] @extension="2015-08-01" (CONF:1198-32551).
  • SHALL contain exactly one [1..1] code (CONF:1198-15405).
    • This code SHALL contain exactly one [1..1] @code="10219-4" Preoperative Diagnosis (CONF:1198-15406).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1198-30863).
  • SHALL contain exactly one [1..1] title (CONF:1198-8099).
  • SHALL contain exactly one [1..1] text (CONF:1198-8100).
  • SHOULD contain zero or one [0..1] entry (CONF:1198-10096) such that it
    • SHALL contain exactly one [1..1] Preoperative Diagnosis (V3) (identifier: 2.16.840.1.113883.10.20.22.4.65) (CONF:1198-15504).
