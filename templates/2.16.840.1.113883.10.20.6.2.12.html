Text Observation [2.16.840.1.113883.10.20.6.2.12] view pdf

Description:

DICOM Template 2000 specifies that Imaging Report Elements of Value Type Text are contained in sections. The Imaging Report Elements are inferred from Basic Diagnostic Imaging Report Observations that consist of image references and measurements (linear, area, volume, and numeric). Text DICOM Imaging Report Elements in this context are mapped to CDA text observations that are section components and are related to the SOP Instance Observations (templateId 2.16.840.1.113883.10.20.6.2.8) or Quantity Measurement Observations (templateId 2.16.840.1.113883.10.20.6.2.14) by the SPRT (Support) act relationship.

A Text Observation is required if the findings in the section text are represented as inferred from SOP Instance Observations.

Contained By: Contains:
Diagnostic Imaging Report (V3)
 SOP Instance Observation
Quantity Measurement Observation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4 STATIC) (CONF:81-9288).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:81-9289).
  • SHALL contain exactly one [1..1] templateId (CONF:81-9290) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.6.2.12" (CONF:81-10534).
  • SHALL contain exactly one [1..1] code (CONF:81-9291).
  • MAY contain zero or one [0..1] text (CONF:81-9295).
    • The text, if present, SHOULD contain zero or one [0..1] reference (CONF:81-15938).
      • The reference, if present, SHOULD contain zero or one [0..1] @value (CONF:81-15939).

        • This reference/@value SHALL begin with a '#' and SHALL point to its corresponding narrative (using the approach defined in CDA Release 2, section 4.3.5.1) (CONF:81-15940).

  • SHOULD contain zero or one [0..1] effectiveTime (CONF:81-9294).
  • SHALL contain exactly one [1..1] value with @xsi:type="ED" (CONF:81-9292).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9298) such that it
    • SHALL contain exactly one [1..1] @typeCode="SPRT" Has Support (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9299).
    • SHALL contain exactly one [1..1] SOP Instance Observation (identifier: 2.16.840.1.113883.10.20.6.2.8) (CONF:81-15941).
  • MAY contain zero or more [0..*] entryRelationship (CONF:81-9301) such that it
    • SHALL contain exactly one [1..1] @typeCode="SPRT" Has Support (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:81-9302).
    • SHALL contain exactly one [1..1] Quantity Measurement Observation (identifier: 2.16.840.1.113883.10.20.6.2.14) (CONF:81-15942).
