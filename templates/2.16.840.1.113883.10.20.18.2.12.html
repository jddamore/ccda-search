Procedure Disposition Section [2.16.840.1.113883.10.20.18.2.12] view pdf

Description:

The Procedure Disposition Section records the status and condition of the patient at the completion of the procedure or surgery. It often also states where the patient was transferred to for the next level of care.

Contained By: Contains:
Procedure Note (V3)
Operative Note (V3)
 none

Conformance Statements:

  • SHALL contain exactly one [1..1] templateId (CONF:81-8070) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.18.2.12" (CONF:81-10466).
  • SHALL contain exactly one [1..1] code (CONF:81-15413).
    • This code SHALL contain exactly one [1..1] @code="59775-7" Procedure Disposition (CONF:81-15414).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:81-26490).
  • SHALL contain exactly one [1..1] title (CONF:81-8072).
  • SHALL contain exactly one [1..1] text (CONF:81-8073).
