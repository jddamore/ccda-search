Functional Status Organizer (V2) [2.16.840.1.113883.10.20.22.4.66, release 2014-06-09] view pdf

Description:

This template groups related functional status observations into categories (e.g., mobility, self-care).

Contained By: Contains:
Functional Status Section (V2)
 Functional Status Observation (V2)
Self-Care Activities (ADL and IADL)
Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="CLUSTER" Cluster (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6) (CONF:1098-14355).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-14357).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-14361) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.66" (CONF:1098-14362).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32569).
  • SHALL contain at least one [1..*] id (CONF:1098-14363).
  • SHALL contain exactly one [1..1] code (CONF:1098-14364).

    • SHOULD be selected from ICF (codeSystem 2.16.840.1.113883.6.254) OR LOINC (2.16.840.1.113883.6.1) (CONF:1098-31417).

  • SHALL contain exactly one [1..1] statusCode (CONF:1098-14358).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1098-31434).
  • SHOULD contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-31585).
  • SHALL contain at least one [1..*] component (CONF:1098-14359) such that it
    • SHALL contain exactly one [1..1] Functional Status Observation (V2) (identifier: 2.16.840.1.113883.10.20.22.4.67) (CONF:1098-14368).
  • SHALL contain at least one [1..*] component (CONF:1098-31432) such that it
    • SHALL contain exactly one [1..1] Self-Care Activities (ADL and IADL) (identifier: 2.16.840.1.113883.10.20.22.4.128) (CONF:1098-31433).
