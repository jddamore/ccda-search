Reaction Observation (V2) [2.16.840.1.113883.10.20.22.4.9, release 2014-06-09] view pdf

Description:

This clinical statement represents the response to an undesired symptom, finding, etc. due to administered or exposed substance. A reaction can be defined described with respect to its severity, and can have been treated by one or more interventions.

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="OBS" Observation (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6 STATIC) (CONF:1098-7325).
  • SHALL contain exactly one [1..1] @moodCode="EVN" Event (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001 STATIC) (CONF:1098-7326).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-7323) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.9" (CONF:1098-10523).
    • SHALL contain exactly one [1..1] @extension="2014-06-09" (CONF:1098-32504).
  • SHALL contain at least one [1..*] id (CONF:1098-7329).
  • SHALL contain exactly one [1..1] code (CONF:1098-16851).
    • This code SHALL contain exactly one [1..1] @code="ASSERTION" (CONF:1098-31124).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.5.4" (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:1098-32169).
  • SHALL contain exactly one [1..1] statusCode (CONF:1098-7328).
    • This statusCode SHALL contain exactly one [1..1] @code="completed" Completed (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14 STATIC) (CONF:1098-19114).
  • SHOULD contain zero or one [0..1] effectiveTime (CONF:1098-7332).
    • The effectiveTime, if present, SHOULD contain zero or one [0..1] low (CONF:1098-7333).
    • The effectiveTime, if present, SHOULD contain zero or one [0..1] high (CONF:1098-7334).
  • SHALL contain exactly one [1..1] value with @xsi:type="CD", where the code SHALL be selected from ValueSet Problem 2.16.840.1.113883.3.88.12.3221.7.4 DYNAMIC (CONF:1098-7335).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-7337) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" Has reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-7338).
    • SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:1098-7343).
    • SHALL contain exactly one [1..1] Procedure Activity Procedure (V2) (identifier: 2.16.840.1.113883.10.20.22.4.14) (CONF:1098-15920).
  • MAY contain zero or more [0..*] entryRelationship (CONF:1098-7340) such that it
    • SHALL contain exactly one [1..1] @typeCode="RSON" Has reason (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-7341).
    • SHALL contain exactly one [1..1] @inversionInd="true" True (CONF:1098-7344).
    • SHALL contain exactly one [1..1] Medication Activity (V2) (identifier: 2.16.840.1.113883.10.20.22.4.16) (CONF:1098-15921).
  • MAY contain zero or one [0..1] entryRelationship (CONF:1098-7580) such that it
    • SHALL contain exactly one [1..1] @typeCode="SUBJ" Has subject (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002 STATIC) (CONF:1098-7581).
    • SHALL contain exactly one [1..1] @inversionInd="true" TRUE (CONF:1098-10375).
    • SHALL contain exactly one [1..1] Severity Observation (V2) (identifier: 2.16.840.1.113883.10.20.22.4.8) (CONF:1098-15922).
