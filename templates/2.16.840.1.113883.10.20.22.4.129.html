Planned Coverage [2.16.840.1.113883.10.20.22.4.129] view pdf

Description:

This template represents the insurance coverage intended to cover an act or procedure.

Contained By: Contains:
Planned Procedure (V2)
Planned Observation (V2)
Planned Supply (V2)
 Author Participation

Conformance Statements:

  • SHALL contain exactly one [1..1] @classCode="ACT" act (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:1098-31945).
  • SHALL contain exactly one [1..1] @moodCode="INT" Intent (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001) (CONF:1098-31946).
  • SHALL contain exactly one [1..1] templateId (CONF:1098-31947) such that it
    • SHALL contain exactly one [1..1] @root="2.16.840.1.113883.10.20.22.4.129" (CONF:1098-31948).
  • SHALL contain at least one [1..*] id (CONF:1098-31950).
  • SHALL contain exactly one [1..1] code (CONF:1098-31951).
    • This code SHALL contain exactly one [1..1] @code="48768-6" Payment Sources (CONF:1098-31952).
    • This code SHALL contain exactly one [1..1] @codeSystem="2.16.840.1.113883.6.1" (CodeSystem: LOINC 2.16.840.1.113883.6.1) (CONF:1098-31953).
  • SHALL contain exactly one [1..1] statusCode (CONF:1098-31954).
    • This statusCode SHALL contain exactly one [1..1] @code="active" Active (CodeSystem: HL7ActCode 2.16.840.1.113883.5.4) (CONF:1098-31955).
  • MAY contain zero or more [0..*] Author Participation (identifier: 2.16.840.1.113883.10.20.22.4.119) (CONF:1098-32178).
  • SHALL contain exactly one [1..1] entryRelationship (CONF:1098-31967) such that it
    • SHALL contain exactly one [1..1] @typeCode="COMP" has component (CodeSystem: HL7ActRelationshipType 2.16.840.1.113883.5.1002) (CONF:1098-31968).
    • SHALL contain exactly one [1..1] act (CONF:1098-31969).
      • This act SHALL contain exactly one [1..1] @classCode="ACT" ACT (CodeSystem: HL7ActClass 2.16.840.1.113883.5.6) (CONF:1098-31970).
      • This act SHALL contain exactly one [1..1] @moodCode="INT" intent (CodeSystem: HL7ActMood 2.16.840.1.113883.5.1001) (CONF:1098-31971).
      • This act SHALL contain at least one [1..*] id (CONF:1098-31972).
      • This act SHALL contain exactly one [1..1] code, which SHALL be selected from ValueSet Payer 2.16.840.1.114222.4.11.3591 DYNAMIC (CONF:1098-31973).
      • This act SHALL contain exactly one [1..1] statusCode (CONF:1098-31974).
        • This statusCode SHALL contain exactly one [1..1] @code="active" Active (CodeSystem: HL7ActStatus 2.16.840.1.113883.5.14) (CONF:1098-31975).
